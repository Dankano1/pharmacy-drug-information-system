# Pharmacy Drug Information System

A structured pharmacy-focused project designed to store and manage
drug information including indications, dosing, contraindications,
interactions, and safety warnings.

## Purpose
This project is intended for:
- Pharmacists
- Pharmacy students
- Clinical decision-support learning
- Health-tech research and prototyping

⚠️ **Disclaimer:**  
This project is for educational and informational purposes only and
does not replace professional medical judgment.

## Features
- Drug indications and dosing
- Contraindications and warnings
- Drug–drug interaction checks
- Expandable data structure

## Project Structure
- `src/` – Core logic
- `data/` – Drug datasets
- `docs/` – Clinical notes and references
- `tests/` – Unit tests
